Q1.Write the difference between HTML tags and HTML attributes?

S.NO HTML tags Attributes
1 HTML tags are used to hold the HTML element. HTML attributes are used to describe the characteristic of an HTML element in detail.
2 HTML tag starts with < and ends with > HTML attributes are found only in the starting tag.
3 HTML tags are almost like keywords where every single tag has unique meaning. HTML attributes specify various additional properties to the existing HTML element.
